Writer User Manual

Toolbar limitations: The toolbar buttons (bold, italic, underline, etc.) cannot format already-selected text. Toggle buttons apply the style only before you start typing — the transformation applies to the next characters you type, not to existing selected text.

What the toolbar can do: Paragraph styles (H1, H2, H3, Normal) and Font selection work on existing blocks regardless.

How to publish: Click Copy HTML. Open War Era news editor. Paste into the body. War Era's editor preserves the formatting.

Entity @mentions: Type @ followed by a name (e.g. @r) to search for War Era entities (countries, regions, companies, military units, battles, alliances, users, articles). The popup filters as you type — spaces are allowed, so @United States works. Use Tab or Enter to select the highlighted item, or click it with the mouse. Once resolved, the entity is cached so subsequent lookups are instant. You can also paste a War Era URL directly into the editor — it will resolve into an @mention automatically.
